# Netflix User Behavior Analysis

## 📌 Project Overview
This project analyzes Netflix user behavior and predicts watch time using machine learning.

## 🎯 Objectives
- Analyze user watch patterns
- Visualize data using graphs
- Build Linear Regression model

## 📊 Features
- Data Cleaning
- EDA (Bar, Histogram, Heatmap, Pie, Scatter)
- Regression Model

## 🔧 Tech Stack
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

## 📈 Model
Linear Regression with evaluation metrics:
- MSE
- MAE
- R2 Score
